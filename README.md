# FTB Backup

This is a backup tools for minecraft Ftb backup. It can upload the latest backup to a remote server (only for webdav for now).

## requirements

- go 1.23

## Installation

```bash
# install
make install

# unsiatll
make uninstall
```

install position: /etc/ftbbackup

## Configuration

position: /etc/ftbbackup/.env
